# Uazapi
### Integração com o Whatsapp usando api UazapiGO - WhatsApp API (v2.0)

Este projeto é uma implementação de funções do Whatsapp utilizando Python e api do uazapi v2.0. Essa api permite realizar ações como envio de mensagens e configurar webhook para diversos eventos.

### Descrição

O objetivo desse projeto é integrar um software com o Whatsapp por meio da api do uazapi. o projeto inclui funções de criação e conexão de instâncias, envio de mensagem,mensagem em massa,criação de grupos e comunidades e muito mais.
